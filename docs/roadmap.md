# ECN2026 – Project Roadmap

This roadmap outlines the planned technical phases of the ECN2026 project.
It reflects a realistic learning and implementation sequence.

## Phase 1 – Core Connectivity
- IP addressing & subnetting
- Static routing and default routes
- Basic firewall rules (allow/deny)
- Connectivity validation

## Phase 2 – Segmentation & Multi-Site
- VLAN design and implementation
- Inter-VLAN routing
- Site-to-site VPN tunnels
- Logical network separation

## Phase 3 – Security & Operations
- Firewall hardening principles
- Monitoring and logging concepts
- Failover testing scenarios
- Documentation and review
